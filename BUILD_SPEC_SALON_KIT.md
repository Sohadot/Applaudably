# مواصفات بناء — Luxury Salon Social Media Kit (منتج الإطلاق الأول)

موجَّه لمصمّمة ماهرة تنقصها فقط معرفة مجال قوالب الإنستغرام.
كل قالب موصوف بـ: **ماذا فيه + المقاس + الهدف منه**. النصوص داخل القوالب **بالإنجليزية**.
المبدأ الذهبي: كل قالب **عام (generic) وقابل للتعديل** — اسم الصالون والأسعار والصور كلها Placeholders يغيّرها المشتري.

---

## أولاً: أساسيات المجال (اقرئيها مرة واحدة)

### المقاسات (px)
| النوع | المقاس | الاستخدام |
|---|---|---|
| منشور فيد (Feed post) | **1080 × 1350** (عمودي 4:5) | الأفضل انتشاراً — استخدميه لكل المنشورات |
| ستوري (Story) | **1080 × 1920** (9:16) | للحجز، العدّادات، التفاعل |
| غلاف هايلايت (Highlight cover) | **1080 × 1920** | أيقونة تظهر في دائرة صغيرة أعلى البروفايل |

### القاعدة الأهم — لماذا يشتري الناس هذه الكيتات
صاحب الصالون **ليس مصمّماً وليس لديه وقت.** يشتري الكيت ليكتب اسمه وأسعاره ويطبع صوره **في دقائق** فينشر محتوى يبدو احترافياً. لذلك:
1. **قابل للتعديل:** كل نص = طبقة نص. كل صورة = داخل Frame.
2. **عام:** لا اسم صالون حقيقي، لا سعر حقيقي — كله أمثلة (`Your Salon Name` / `$00`).
3. **نظيف:** لا مؤشر فأرة، لا نص Canva افتراضي (`123 Anywhere St.`)، لا حروف مشوّشة. راجعي كل قالب قبل التصدير.

### الهوية البصرية (Quiet Luxury للصالونات الراقية)
جمهورنا صالونات **راقية** — والفخامة الهادئة تعني: محايد أكثر، ذهبي **بحذر**، وردي **خافت** لا صريح (الوردي الفاقع يوحي بصالون أظافر للمراهقات، لا سبا راقٍ).

| اللون | HEX | الاستخدام |
|---|---|---|
| Warm Cream | `#F3EDE4` | خلفية أساسية |
| Greige | `#E7DED2` | خلفية ثانوية / بلوكات |
| Dusty Rose | `#D8B7AD` | لمسة وردية خافتة (لا تُكثري منها) |
| Gold | `#B89B6E` | خطوط رفيعة وتفاصيل فقط — ليس مساحات |
| Espresso | `#3A322C` | نص أساسي |
| Taupe | `#7A6E63` | نص ثانوي |

**الخطوط:** العناوين Cormorant Garamond · النص Montserrat (Light للفقرات، Medium للتسميات). خط واحد للعناوين والتزمي به.

## المجموعة 1 — قوائم الأسعار والخدمات (10 منشورات فيد) ⭐ قلب الكيت
هذه **أهم سبب** لشراء كيت صالون. اجعليها أنظف وأجمل ما في الكيت.

1. **قائمة خدمات كاملة** — عمود واحد: اسم الخدمة يساراً … نقاط تنقيط … السعر يميناً، ‏8–10 صفوف، عنوان "SERVICE MENU" + `Your Salon Name` + خط ذهبي رفيع فاصل.
2. **قائمة عمودين** — 3 فئات (Hair / Color / Treatments)، تحت كلٍّ 3–4 خدمات بأسعارها.
3. **الخدمات المميّزة** — 3 خدمات فاخرة في بطاقات، كل واحدة بوصف سطر + "from $00".
4. **قائمة أسعار الصبغة** — Balayage / Highlights / Root touch-up / Toner / Gloss مع نطاقات أسعار.
5. **باقات المناسبات** — 3 باقات (Bridal / Party / Photoshoot) مع "ما تتضمّنه" + السعر.
6. **عرض العميل الجديد** — بطاقة "First Visit" بسعر خاص.
7. **الإضافات السريعة** — Add-ons (Deep condition, Scalp treatment, Blow-dry) بأسعار صغيرة.
8. **بطاقة اشتراك/باقة ولاء** — "5 Blow-drys for $00".
9. **خدمة واحدة مميّزة** — خدمة واحدة كبيرة ("BALAYAGE — $00") مع إطار صورة أنيق.
10. **قائمة موسمية** — عنوان قابل للتغيير ("Autumn Menu") بخدمات محدودة.

## المجموعة 2 — العروض الترويجية (12 منشور فيد)
11. خصم نسبة — "20% OFF" كبيرة + الخدمة + كود + تواريخ الصلاحية + سطر شروط صغير.
12. عرض خاطف — "48 HOURS ONLY" (إلحاح).
13. رشّحي صديقة — "Refer a friend, you both get $00 off".
14. خدمة جديدة — "NOW OFFERING [Service]".
15. عرض موسمي — "Holiday Glow" بطابع المناسبة.
16. بطاقات الهدايا — "Gift the gift of self-care — Gift cards available".
17. مكافأة ولاء — "Your 6th visit is on us".
18. عرض منتصف الأسبوع — "Midweek Special — [Day] only".
19. باقة مجمّعة — "Cut + Color + Treatment = $00 (save $00)".
20. عرض عيد الميلاد — "It's your birthday month — enjoy…".
21. مواعيد اللحظة الأخيرة — "Same-day availability — DM to book".
22. ذكرى/إنجاز — "Celebrating 5 years — specials all week".

## المجموعة 3 — الحجز والمواعيد (10 ستوريز 1080×1920)
23. "Now booking [Month]" — إعلان توفّر.
24. "Book now" — خطوات: DM / Link in bio / Call.
25. "A few spots left this week" — إلحاح.
26. "Fully booked — join the waitlist".
27. "How to book" — دليل 3 خطوات.
28. "Our hours" — بطاقة ساعات العمل الأسبوعية.
29. "Cancellation policy" — بطاقة سياسة واضحة (الصالونات تحتاجها كثيراً).
30. "What to expect at your visit" — دليل الزيارة الأولى.
31. "New client? Start here" — حجز + استقبال.
32. "Appointment reminder" — ستوري تذكير عام.

## المجموعة 4 — التقييمات (8 منشورات فيد)
33. اقتباس واحد + 5 نجوم + اسم العميلة الأول.
34. اقتباس فوق إطار صورة (بطبقة تعتيم خفيفة).
35. "Review of the week" — بطاقة مميّزة.
36. بطاقة تقييم بأسلوب "screenshot" أنيق.
37. تقييمان في تصميم مقسوم.
38. عرض التقييم — "5.0 ★ · 200+ happy clients".
39. "Client love" — اقتباس + إطار صورة العميلة.
40. بطاقة على طراز مراجعة Google/Booking.

## المجموعة 5 — قبل / بعد (6 منشورات فيد)
41. جنباً إلى جنب بتسميتَي "BEFORE / AFTER".
42. تقسيم قطري (slider).
43. تحوّل مع اسم الخدمة والمدة.
44. تحوّل لوني (Color transformation).
45. ثلاث مراحل تدريجية.
46. قبل/بعد مع اقتباس العميلة.

## المجموعة 6 — أغلفة الهايلايت (10، مقاس 1080×1920، العنصر في الوسط ليظهر بالدائرة)
أيقونة خط ذهبي واحدة + كلمة، على خلفية Cream:
`Services` · `Prices` · `Booking` · `Reviews` · `Before/After` · `Team` · `Location` · `Hours` · `Offers` · `FAQ`

## المجموعة 7 — دليل + غلاف (للتغليف والـListing)
- **دليل الألوان والخطوط** (صفحة): الألوان الستة بدوائر وأكوادها + الخطوط بأمثلة.
- **غلاف واحد فقط** للكيت (اعتمدي غلاف "REFINED" المجمّع — كان الأقوى، بعد حذف عنوان `123 Anywhere St.`).

## التسليم (بعد اكتمال القوالب)
1. لكل ملف Canva: **Share → Template link → Copy**.
2. ملف `Quick Start Guide` (PDF، 3 صفحات): غلاف + الروابط كأزرار + خطوات البدء وسطر الترخيص:
   `License: one salon (yours or a client's). No resale or link sharing.`
3. نزّلي **PDF Standard** — هو الملف الوحيد المرفوع على Etsy.
