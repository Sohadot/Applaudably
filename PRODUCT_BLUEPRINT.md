# Product Blueprint

Every Applaudably product is defined with this template **before** any design work starts in Canva.
A product without a completed blueprint does not get built, listed, or shown on the site.

## The Template

```
## Product Name
## Target Buyer
## Use Case
## Template Count
## Included Formats
## Content Categories
## Visual Direction
## Etsy Title
## Etsy Tags
## Listing Images Needed
## Status            (blueprint / in production / listed / live on site)
```
